# 🏦 Banking Voice Bot

An AI-powered Banking Voice Bot built with 
Google Dialogflow for customer service automation.

## 🎯 Features
- Check account balance (savings/current/credit)
- Block/unblock debit & credit cards
- Transfer money between accounts
- Locate nearest ATM
- Smart fallback handling
- Multi-turn conversations with slot filling

## 🛠️ Tech Stack
- Google Dialogflow ES
- Natural Language Processing (NLP)
- Intent & Entity Management
- Conversational AI

## 💬 Sample Conversations
**User:** Hello  
**Bot:** Welcome to Banking Bot! How can I assist you?

**User:** Check my balance  
**Bot:** Which account - savings, current or credit?

**User:** My card got stolen  
**Bot:** Your card has been blocked successfully.

## 🚀 How to Use
1. Clone this repository
2. Import agent.zip into Google Dialogflow
3. Test in Dialogflow console
